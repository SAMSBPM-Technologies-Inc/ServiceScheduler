# Vendor Worker Guide

This guide covers what a Vendor Worker can see and do in the ServStack vendor portal.

**URL:** `https://servstack.ca/vendor`

Workers have **read-only** access. They can view all operational data but cannot make changes to products, plans, settings, or subscriptions.

---

## What Workers Can See

| Section | Worker Access |
|---------|---------------|
| Dashboard | View stats and recent alerts |
| Products | View product list |
| Plans | View plans and their configuration |
| Subscriptions | View all subscriptions and their details |
| Payments | View all payment records |
| Alerts | View all alerts |
| Team | View team members list |
| Settings | Not accessible |

---

## Dashboard

The dashboard shows:

- **Active subscriptions** — current active count
- **Revenue this month** — total paid this calendar month
- **Pending payments** — awaiting completion
- **Recent alerts** — latest notifications

This is a live snapshot of the business.

---

## Subscriptions

Workers can view all subscriptions and their full detail:

- Customer name and contact
- Plan name and type
- Current status (Active / Paused / Cancelled)
- Product selections (Fixed) or task schedules (Configurable)
- Instructions/notes
- Payment history for the subscription

Workers **cannot** change subscription status, edit selections, or add instructions.

---

## Payments

Workers can view all payment records including:

- Amount and billing period
- Status (Pending / Paid / Failed / Refunded)
- Customer and subscription reference
- Stripe payment intent ID

Workers cannot issue refunds or modify payment records.

---

## Alerts

Workers can see all alerts raised for the vendor. This is useful for monitoring delivery issues, renewal reminders, and stock alerts.

Workers cannot mark alerts as read or manage alert rules.

---

## What Workers Cannot Do

- Add, edit, or deactivate products
- Create, edit, or activate plans
- Change subscription status (pause, resume, cancel)
- Edit customer selections or schedules
- Invite or manage team members
- Change Stripe keys or settings
- Access vendor settings

If you need to make a change, contact your Vendor Admin.

---

## Logging In

1. Go to `https://servstack.ca/vendor`
2. Enter the email and password provided by your Vendor Admin
3. You are taken to the dashboard

If you cannot log in or have been deactivated, contact your Vendor Admin.
